# Django React Demo

# Installation
1. `git clone https://github.com/mic159/django-react-demo.git`
2. `cd django-react-demo`
3. `virtualenv .env`
4. `pip install -e .`
5. `yarn install`
6. `yarn run build-client`
7. `yarn run build-server`
8. `npm run react-service &` (as a background task)
9. `demo runserver`
